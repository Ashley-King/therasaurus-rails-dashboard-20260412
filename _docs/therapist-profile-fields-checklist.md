# Therapist Profile Page - Fields Checklist

Use [therapist-profile-design-brief.md](therapist-profile-design-brief.md) as the source of truth.

This checklist exists only to point readers to the regenerated designer field inventory. The older checklist duplicated the brief and contained outdated public-profile assumptions.

The current rules are:

- No embedded maps.
- "View on map" appears only when a street address is public.
- Every public address shows at least city, state, and ZIP.
- Email addresses are never shown.
- Phone-call links are not shown when the therapist hides their phone number.
