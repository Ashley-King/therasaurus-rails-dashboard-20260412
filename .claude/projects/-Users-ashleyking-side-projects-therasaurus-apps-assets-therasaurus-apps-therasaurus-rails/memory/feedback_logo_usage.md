name: Logo usage rules
description: Which logo SVG variants to use on light vs dark backgrounds
type: feedback
---

Use `logo-main-stacked.svg` (saved as `logo.svg`) on light backgrounds. Use `logo-stacked-white.svg` (saved as `logo-white.svg`) on dark backgrounds. Do not use horizontal wordmark or grey variants.

**Why:** User corrected incorrect logo choice — the stacked logo with the dinosaur is the primary brand mark.
**How to apply:** When adding the logo to any layout or component, pick the variant based on background color.
