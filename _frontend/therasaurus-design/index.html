What Changed in v2.0

Consolidated Background Colors

Reduced from 5 near-identical light neutrals (#F3F1F6, #F4F1F6, #F5F5F5, #FAFAFA, #F0F4FC) down to 3 distinct tiers: White (cards), #FAFAFA (page background), and #F3F1F6 (branded sections).

Clear Text Color Hierarchy

Added a 7-step gray hierarchy (matching Paperbell's approach) from #0A0A0A (headings) through #D4D4D4 (disabled). Each step has a defined purpose so text weight decisions are predictable.

Simplified Type Scale

Removed the redundant 25px size (was too close to 24px). Dropped extrabold (800) weight. All headings now use bold (700) for better readability at large sizes. Clean scale: 12, 14, 16, 18, 24, 30, 48.

Two Fonts Only

Removed the system font stack and Times fallback. All rendering now uses Poppins (headings) or Roboto (everything else). No ambiguity about which font appears where.

Consistent Button System

Three tiers: Primary (plum bg, white text, 6px radius), Outline (white bg, plum text, 8px radius), and CTA (dark plum). Hero buttons keep the pill shape as a special case. All buttons use Roboto semibold.

Standardized Border Radius

Removed the 10px "base" radius and 24px "3xl" radius. Inputs and buttons use 6px (md). Cards use 12px (xl). This matches Paperbell's tighter radius system while keeping things clean.

Added Missing Component Styles

Added sidebar navigation, dropdown menus, tabs, alert banners, and status badges that were missing from v1. These follow the same structural patterns as Paperbell's components.

Brand-Aligned Chart Colors

Reordered chart palette to lead with plum and teal (brand colors) instead of orange. Creates a more cohesive data visualization identity.
